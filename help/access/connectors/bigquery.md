# BigQuery

Provides access to [Google BigQuery](https://cloud.google.com/bigquery/) database
using SQL queries via JDBC driver. 

## Connection Parameters

```json
{
  "conn. string": "",
  "project_id": "",
  "credentials" : {
    "parameters": {
      "login": "",
      "password": ""
    }
  }
}
```
