# S3

Provides access to [S3](https://aws.amazon.com/s3/) as [file share](files.md).

## Connection Parameters

```json
{
  "parameters": {
    "region": "us-east-1",
    "bucket": "",
    "dir": "",
    "encryption": "",
  },
  "credentials": {
    "parameters": {
      "accessKey": "",
      "secretKey": ""
    }
  },
}
```
