# Impala

Provides access to [Impala](https://impala.apache.org/) database
using SQL queries via JDBC driver. 

## Connection Parameters

```json
{
  "server": "",
  "port": 21050,
  "db": "",
  "schema": "",
  "conn. string": "",
  "credentials" : {
    "parameters": {
      "login": "",
      "password": ""
    }
  }
}
```
